# movieSearch
Basic Movie Search, showing how to use API in webapp.

## Finished
![](finishedApp.gif)

## How to Use - Installation
* Obtain API Key from [OMDb API](http://www.omdbapi.com)
* Clone this repo.
* Install [Node.js](https://nodejs.org/en/)
* Go to source directory, where you cloned.
   * Run ```npm install```
   * Run ```node app.js```
